📸 Berimbolo Security Website

A simple and responsive web project for showcasing smart security products such as cameras, doorbells, smart locks, and security lighting.

🚀 Project Overview

This project is a static website built using HTML, CSS, and JavaScript. It presents a modern UI for a security company, allowing users to explore different products and services.

🧰 Technologies Used
HTML5
CSS3 (including SCSS)
JavaScript
Bootstrap
jQuery
Font Awesome


📁 Project Structure
Berimbolo-main/
│
├── index.html          # Main webpage
│
├── css/
│   ├── style.css       # Main styling
│   ├── responsive.css  # Responsive design
│   ├── bootstrap.css   # Bootstrap framework
│   └── images/         # CSS-related images
│
├── js/
│   ├── index.js        # Main JS logic
│   ├── custom.js       # Custom scripts
│   ├── bootstrap.js    # Bootstrap JS
│   └── jquery-3.4.1.min.js
│
├── images/             # Website images
│
└── fonts/              # Font Awesome fonts


💻 How to Run the Project
Download or extract the project files
Open the folder
Double-click on index.html
The website will open in your browser

✅ No installation or server required

✨ Features
Responsive design (works on mobile & desktop)
Modern UI with Bootstrap
Product showcase (cameras, smart locks, doorbells, etc.)
Image-based sections for better visualization
Smooth layout and navigation
📌 Notes
Make sure all files stay in their correct folders (CSS, JS, images)
Internet connection may be needed for some fonts/icons
You can edit index.html and style.css to customize the website
